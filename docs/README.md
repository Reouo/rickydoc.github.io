# 星尘编年史

> — 记忆编织者的旅程 —

## 项目简介

「星尘编年史」是一款基于 Web 的沉浸式文字冒险游戏。在一个魔法与科技交织的世界中，你将扮演一名年轻的"记忆编织者"，穿越不同的记忆碎片，修复被黑暗侵蚀的世界线。

## 游戏特色

| 特色 | 描述 |
|------|------|
| 📖 沉浸式叙事 | 精心设计的剧情故事，配合打字机效果呈现 |
| 🔀 多分支剧情 | 每个选择都会影响故事走向和角色属性 |
| 💾 存档系统 | 支持 3 个存档槽位，随时保存/加载进度 |
| 🏆 成就收集 | 8 个隐藏成就，记录你的冒险历程 |
| 📤 存档导入导出 | 支持存档代码的导入导出，方便分享 |
| ✨ 精美界面 | 星空粒子特效、流畅动画、响应式设计 |

## 游戏属性

角色拥有三项核心属性，不同的选择会影响属性变化：

- 🧠 **智慧** - 影响分析与判断能力
- ⚔️ **勇气** - 影响行动与战斗能力  
- 💜 **同理心** - 影响与他人的情感连接

## 故事背景

千年前，星尘之力维持着世界的平衡。但一场大战撕裂了时空，记忆碎片散落各处，黑暗开始蔓延。"编织者"是唯一能够修复这一切的人，而你...是最后的编织者。

### 第一幕：遗忘森林

你从星尘圣殿苏醒，接受了长老艾尔文的使命，前往遗忘森林寻找第一块记忆碎片。在那里，你遇到了迷路的小女孩小雨，帮助她寻找失散的姐姐...

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- 无任何外部依赖
- 支持移动端响应式布局
